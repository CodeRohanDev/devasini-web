# Devasini Family Office Website

Professional website for Devasini Family Office - institutional-grade wealth management and cybersecurity services.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
src/
├── app/                    # Next.js App Router pages
│   ├── about/

│   ├── contact/

│   ├── leadership/
│   ├── services/
```

## Pages

- **Home** - Hero, value proposition, services overview, leadership highlight
- **About** - Company profile, philosophy, security standards
- **Services** - Family office & cybersecurity packages
- **Leadership** - Core team and advisory board
- **Events** - Upcoming and past events, partnerships
- **Athletes** - Athletes program and legacy initiatives
- **Contact** - Secure contact form
